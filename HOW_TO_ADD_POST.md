# 📝 如何添加新文章

## 快速步骤

### 1️⃣ 复制模板

```bash
cp public/posts/_template.md public/posts/6-你的文章标识.md
```

### 2️⃣ 编辑文章

打开新创建的文件，修改 Front Matter：

```markdown
---
id: 6                                    # 改成新的唯一 ID
title: "你的文章标题"                     # 修改标题
category: "前端开发"                      # 选择分类：算法 或 前端开发
date: "2024-12-24"                       # 修改日期
excerpt: "文章摘要..."                    # 写一段摘要
tags: ["标签1", "标签2"]                 # 修改标签
readTime: "10 分钟"                      # 预估阅读时间
---

# 你的文章标题

开始写作...
```

### 3️⃣ 生成索引

```bash
npm run generate:posts
```

### 4️⃣ 完成！

刷新浏览器查看你的新文章 🎉

## 📋 注意事项

- ✅ ID 必须唯一（建议使用递增数字）
- ✅ 文件名格式：`{id}-{文章标识}.md`
- ✅ 分类只能是 "算法" 或 "前端开发"
- ✅ 日期格式：YYYY-MM-DD
- ✅ 标签用数组表示：`["标签1", "标签2"]`
